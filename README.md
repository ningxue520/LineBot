標題 :
------
- `UA = CHROMEOS\t1.4.17\tChrome_OS\t1`
- `LA = Line/1.4.17`

如何在Termux中取得Line代碼 :
------
- `pkg install nodejs -y`
- `pkg install git -y`
- `git clone https://github.com/ningxue520/linetoken`
- `pkg install nano`
- `cd linetoken`
- `npm install`
- `npm i`
- `npm i -g npm`
- `npm start`

如何再次取得代碼 (主機重啟後) :
------
- `cd linetoken`
- `npm start`
